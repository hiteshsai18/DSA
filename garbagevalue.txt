A garbage value is an unpredictable, random value present in a computer's memory when a variable is declared but not initialized. 
It represents leftover data from previous programs that used the same memory location, leading to undefined behavior

a=10    |  10  | 1000
b=20    |  20  | 1008
c=a+b   |  30  | 1016

In recyclebin;

  | 1000 | 2000
  | 1008 | 2016
  | 1016 | 2032

garbage value is ----> |10162032| which is 30.
